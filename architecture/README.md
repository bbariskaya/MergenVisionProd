# architecture/

This directory is for architecture decision records, diagrams, and high-level design documents.

Architecture docs must follow `AGENTS.md` and `phase0beforestarting.md`.

Allowed for now:

- ADRs
- Mermaid diagrams
- hotpath flow documents
- data-model contracts

Forbidden for now:

- backend source code
- Dockerfiles
- benchmark scripts
- generated engine/model files
